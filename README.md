# Expense-Tracker

A simple command-line expense tracker built with Python.

## Features
- Add expenses with description, amount, and category
- View all expenses with a running total
- View spending broken down by category
- Data is saved locally so nothing is lost on exit

## How to run
1. Make sure Python is installed
2. Clone this repo: `git clone https://github.com/divya-sharma-31/Expense-Tracker.git`
3. Run: `python expense_tracker.py`

## Technologies used
- Python 3
- JSON for data storage

## What I learned

- Reading and writing JSON files
- Building a simple menu-driven CLI app
